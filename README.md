## Built Using
 - Python 3.8.2
 - discord.py 1.3.3
 - PyYAML 5.3.1
 - pony 0.7.13

## Execution
Update `config.yaml` to add your Discord Token and other settings
```bash
$ pip install -r requirements.txt
$ python -m Bot
```
To pull data run
```bash
$ python -m Database
```
